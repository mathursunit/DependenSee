Branding assets for Carrier DependenSee
=======================================

Place the logo here:

  DependenSee.png   <- the Carrier DependenSee logo (PNG, square works best)
  DependenSee.ico   <- optional; generated from the PNG (used as the exe/taskbar icon)

How they're used:
  * DependenSee.png  -> app window icon (fallback), firewall PDF header, user guide.
  * DependenSee.ico  -> embedded exe icon and Start Menu shortcut icon.

To generate the .ico from the .png, run scripts\make-icon.ps1 (Windows),
or ask the tooling to convert it. Both files are copied next to the app at build.
